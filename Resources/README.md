#### Exoplanets API:

**Documentation**: https://exoplanetarchive.ipac.caltech.edu/docs/program_interfaces.html

**API endpoint**:

```http
GET https://exoplanetarchive.ipac.caltech.edu/cgi-bin/nstedAPI/nph-nstedAPI?table=cumulative&where=koi_disposition like 'CANDIDATE'&format=json
```
